# SunC++ - C/C++/C# 编辑器

基于 PyQt6 开发的 C、C++、C# 代码编辑器，支持语法高亮和基本编辑功能。

## 功能特性

- ✅ 支持 C、C++、C# 语法高亮
- ✅ 基本编辑功能（复制、剪切、粘贴、全选）
- ✅ 文件操作（新建、打开、保存、另存为）
- ✅ 支持多种文件扩展名
- ✅ 可手动切换语言模式
- ✅ 状态栏显示状态信息
- ✅ 内置 C++ 编译器支持
- ✅ 编译和运行代码功能
- ✅ 编译错误显示
- ✅ 运行结果显示

## 支持的文件类型

- **C 语言**: `.c`, `.h`
- **C++ 语言**: `.cpp`, `.cc`, `.cxx`, `.hpp`
- **C# 语言**: `.cs`

## 安装依赖

```bash
# 安装 PyQt6
pip install -r requirements.txt
```

## 运行编辑器

```bash
# 进入 SunC++ 目录
cd SunC++

# 运行编辑器
python main.py
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建文件 |
| `Ctrl+O` | 打开文件 |
| `Ctrl+S` | 保存文件 |
| `Ctrl+Shift+S` | 另存为 |
| `Ctrl+Q` | 退出 |
| `Ctrl+C` | 复制 |
| `Ctrl+X` | 剪切 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+A` | 全选 |
| `F7` | 编译 |
| `F5` | 运行 |
| `F9` | 编译并运行 |

## 语言切换

- 通过菜单栏 `语言` -> `C`/`C++`/`C#` 切换语言模式
- 打开文件时会自动根据文件扩展名设置语言

## 界面说明

1. **菜单栏**: 文件、编辑、语言、帮助
2. **工具栏**: 常用操作按钮
3. **编辑区**: 代码编辑区域，支持语法高亮
4. **状态栏**: 显示当前状态信息

## 示例

### 打开 C 文件

```bash
# 运行编辑器后，点击文件 -> 打开，选择 .c 或 .h 文件
# 编辑器会自动识别为 C 语言并应用相应的语法高亮
```

### 打开 C++ 文件

```bash
# 选择 .cpp, .cc, .cxx, .hpp 文件
# 编辑器会自动识别为 C++ 语言
```

### 打开 C# 文件

```bash
# 选择 .cs 文件
# 编辑器会自动识别为 C# 语言
```

## 注意事项

1. 本编辑器为基础版本，仅提供基本编辑功能
2. 支持 UTF-8 编码的文件
3. 语法高亮规则可能需要根据实际需求调整

## 技术实现

- **GUI 框架**: PyQt6
- **语法高亮**: 基于 QSyntaxHighlighter 实现
- **文件操作**: 使用 QFileDialog
- **代码结构**: 面向对象设计
